# 📋 EasyGestion

EasyGestion est une application de gestion simple destinée aux petites entreprises et indépendants.

Elle permet de gérer les clients, les rendez-vous et les factures depuis une interface claire et facile à utiliser.

---

## 🚀 Fonctionnalités

### 👥 Gestion des clients
- Ajouter un client
- Modifier les informations d'un client
- Supprimer un client
- Consulter la liste des clients

### 📅 Gestion des rendez-vous
- Ajouter un rendez-vous
- Choisir un client
- Choisir une date
- Choisir une heure
- Supprimer un rendez-vous

### 🧾 Gestion des factures
- Créer une facture
- Associer une facture à un client
- Enregistrer un montant
- Supprimer une facture

### 📊 Tableau de bord
- Nombre de clients
- Nombre de rendez-vous
- Chiffre d'affaires

---

## 🛠️ Technologies utilisées

- Python
- Streamlit
- SQLite
- Pandas
